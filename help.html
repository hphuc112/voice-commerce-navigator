Getting Started

Basic Navigation

Click the microphone button to start voice control. Try these commands:

Try saying: "Go to products page"
Try saying: "Show my cart"

Voice Command Reference

Navigation Commands

  • "Go home" - Return to main page
  • "Show products" - Browse products
  • "Open cart" - View shopping cart

Cart Management

  • "Add product number [X]" - Add item by number
  • "Remove product [X]" - Remove item
  • "Clear cart" - Empty cart

Product Interactions

  • "Search for [keyword]" - Find products
  • "Show details for product [X]" - View item
  • "Filter [category]" - Narrow results
